This plugin enables you to embed Flash and HTML5 video on your WordPress website.

== Description ==

This plugin makes it extremely easy to deliver Flash and HTML5 video through your WordPress website.  

<strong>Key Features</strong>

* Full support for JPlayer for Flash and HTML5.
* Manage multiple custom players.
* Support for adding external video to your media library including Youtube and RTMP streams.
* A powerful shortcode system for customizations at embed time.
  
== Installation ==

Installing for the first time:

1. Place the plugin folder in your plugin directory.
2. Activate the plugin in the plugin manager
  
== Requirements ==

* WordPress 2.8.6 or higher
* PHP 5.0 or higher

== Usage ==
Use these shortcode

= Audio =
[radium_audio href="file url here" /]

= Video =
[radium_video href="file url here" /]
